# united-domains
United Domains unofficial api

# Install

```python
pip3 install virtualenv
```
```python
virtualenv united-domains
```
```python
source myproject/venv/bin/activate
```

### Starting

Move the *example.env* to *.env* and modify email and username.

### Run via (in src folder)
```python
python3 main.py
```
